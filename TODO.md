# Product Todo List

## GM Settings tab
- Allow players to rename NPCs and allow the GM to toggle the ability on/off.
- Allow players to move NPCs and allow the GM to toggle the ability on/off.
- GM “Blindfold” toggle to hide the map canvas from all players temporarily.

## Drawing tools
- Player drawing: temporary lines visible to all, auto-fade after ~10 seconds.
- GM drawing: lines with fade/persist toggle.
- Player and GM ping feature: blinking icon on the map canvas for a few seconds, rate-limited to one active ping per player.

## Token management
- Resize placed tokens (growth effects, illusions, or grid fit adjustments).
- Token folders for organization.
- Highlight selected PC/NPC from GM list on the map.
- Ability to set token highlight color for status conditions.
- Click-to-edit NPC names on tokens.
- NPC notes popup on token click.
- GM can rename NPCs from NPC tab in GM Controls.

## Map management
- Map folders for organization.

## Health tracking
- PC/NPC health values; GM controls visibility (all players, self-only, or GM-only); show in “NPCs on current map” pane.

## UI/UX
- Display file type and size limits on token upload UI.
- Collapsible player panel (chat/dice/notes).
- Player display settings: primary and accent color pickers.
